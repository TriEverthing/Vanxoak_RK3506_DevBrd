# Vanxoak_RK3506_DevBrd

Vanxoak_RK3506_DevBrd是一个专门为万科奥华的RK3506的邮票孔核心板设计的扩展板。PCB的结构参考了树莓派，不过为例方便Layout扩展排针略有不同。PCB为两层板双面贴设计，满足嘉立创的免费工艺，极大地降低了复刻成本。
